## Scoot Scout

Scoot Scout aggregates the location of six different brands of motorized scooters around Nashville.

Technology Used:
* React.js
* HTML5/CSS
* Javascript
* Postman
* Leaflet (markercluster, geosearch)
* Mapbox

APIs Used:
* Google
* Bird
* Lime
* Spin
* Nashville Metro Open Data Source
