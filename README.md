## Scoot Scout

Tired of opening up five apps to find the nearest motorized scooter? Scoot Scout aggregates multiple scooter brands into one convenient place, so you can get to that concert sooner. 

Technology Used:
* React.js
* HTML/CSS
* Javascript
* Postman
* Leaflet
* Mapbox

APIs Used:
* Google
* Bird
* Lime
* Spin
